### Requerimientos

* [NodeJS](https://nodejs.org/)

### Empezar

```bash

# Clone the repo & cd into it.
git clone https://github.com/PabloVallejo/poli-open.git
cd poli-open

# Install Bower and NPM dependencies.
bower install
npm install

# Build styles and assets.
gulp move_assets
gulp sass

# Watch for .sass files changes.
gulp sass:watch

```
